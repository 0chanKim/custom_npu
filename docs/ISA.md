## 4. 레지스터 맵

### 4.1 제어 레지스터
| 주소 | 이름 | 설명 |
|------|------|------|
| 0x00 | CTRL | 제어 레지스터 |
| 0x04 | STATUS | 상태 레지스터 |
| 0x08 | CONFIG | 설정 레지스터 |

### 4.2 DMA 레지스터
| 주소 | 이름 | 설명 |
|------|------|------|
| 0x10 | SRC_ADDR | 소스 주소 |
| 0x14 | DST_ADDR | 목적지 주소 |
| 0x18 | LENGTH | 전송 길이 |
